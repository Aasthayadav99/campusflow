# 🎓 Campus Flow

Campus Flow is a full-stack college utility platform designed to simplify campus life by bringing essential student services into one place. It provides an intuitive interface for students and faculty to access important information, manage academic activities, and stay updated with campus events.

## 🚀 Features

* 👤 Secure User Authentication
* 📅 Academic Calendar & Event Management
* 📢 Notices & Announcements
* 📚 Course and Subject Information
* 📝 Assignment & Submission Tracking
* 🎯 Attendance Management
* 📊 Student Dashboard
* 👨‍🏫 Faculty Dashboard
* 🔍 Search and Filter Functionality
* 📱 Fully Responsive Design

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/CampusFlow.git
```

### Navigate to the project

```bash
cd CampusFlow
```

### Install dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd ../server
npm install
```

### Configure Environment Variables

Create a `.env` file inside the server folder.

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### Run the application

Backend

```bash
npm run dev
```

Frontend

```bash
npm start
```
